Room Editor

Separate room editor entrypoint. It uses the shared room layout source and supports selection, dragging, renaming, rotation previews, editable walkable surface bounds, collision toggles, GLB-backed model objects, primitive shapes, model-dimension calibration, undo/redo, and JSON import/export. Camera controls match game debug mode: WASD move and arrow keys rotate.

Back To Game
Left-click any room object to select it. Drag on the scene to move it across the floor plane. Rotation previews update live while you drag the slider, and the undo entry is only committed when you release it. Surface-capable objects can be made walkable, obstacle-capable objects can be given collisions, and GLB model objects can calibrate their dimensions from their loaded bounds.
